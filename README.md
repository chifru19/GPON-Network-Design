# GPON Network Design & Simulation for a High-Density Residential Area

## Overview
This repository contains a professional-grade GPON (Gigabit Passive Optical Network) design for a high-density residential deployment. The project adheres to ITU-T G.984 standards and demonstrates end-to-end network planning, from link budget analysis to OLT configuration.

## Project Structure
- **/01_Executive_Summary**: Formal project proposal.
- **/02_Design_Documents**: Network topology and layout diagrams.
- **/03_Calculations**: Link budget and loss analysis (CSV).
- **/04_Configuration_Scripts**: Standardized OLT/ONT configuration templates.
- **/05_Specs_and_Datasheets**: Reference documentation.
