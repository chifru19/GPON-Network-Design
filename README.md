# GPON Network Design & Simulation for a High-Density Residential Area

## Overview
This repository contains a professional-grade GPON design for high-density residential deployment and a strategic Connectivity Resilience proposal for enterprise-grade network stability. The project adheres to ITU-T G.984 standards and integrates business-critical resilience metrics.

## Project Structure
- **/01_Executive_Summary**: 
  - *proposal.md*: Technical network design proposal.
  - *connectivity_resilience.md*: Business-focused resilience and compliance strategy.
- **/02_Design_Documents**: Network topology and layout diagrams.
- **/03_Calculations**: Link budget and loss analysis (CSV).
- **/04_Configuration_Scripts**: Standardized OLT/ONT configuration templates.
- **/05_Specs_and_Datasheets**: Reference documentation.
